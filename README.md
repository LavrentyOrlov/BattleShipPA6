# Project Name: Battleship

## Project Info
- Date: November 2025
- Created by: Lavrenty "Larry" Orlov

## Overview
Description: This is a text-based version of a board game called Battleship, played between a player and
the computer. This program's capabilities include processing user input in the terminal window, random number generation, mathematical computations, and displaying output results both in the terminal window and in an output stream file.

## Features
- Controls the complete gameplay of Battleship, displays rules, features continuous user interaction with the terminal, tests for numerical edge cases, and logs data saved to an external file.
- Modular design with separate '.c' and '.h' files
- Easy to run in Visual Studio

## Tech Stack
- Language: C
- IDE: Visual Studio Community

## How to Run
1. Clone the repository on your device's command-line interface:
- ```git clone https://github.com/lavrentyorlov/BattleShipPA6.git```
2. Open the project in Visual Studio.
3. Build the solution.
4. Run the program.
